# `kolb-bot tui`

Open the terminal UI connected to the Gateway.

Related:

- TUI guide: [TUI](/web/tui)

## Examples

```bash
kolb-bot tui
kolb-bot tui --url ws://127.0.0.1:18789 --token <token>
kolb-bot tui --session main --deliver
```
