# `kolb-bot setup`

Initialize `~/.kolb-bot/kolb-bot.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
kolb-bot setup
kolb-bot setup --workspace ~/.kolb-bot/workspace
```

To run the wizard via setup:

```bash
kolb-bot setup --wizard
```
