# SOUL Evil Hook

Small persona swap hook for KolbBot.

Docs: https://github.com/kolbick/Kolb-Bot/blob/main/docs/hooks/soul-evil

## Setup

1. `kolb-bot hooks enable soul-evil`
2. Create `SOUL_EVIL.md` next to `SOUL.md` in your agent workspace
3. Configure `hooks.internal.entries.soul-evil` (see docs)
